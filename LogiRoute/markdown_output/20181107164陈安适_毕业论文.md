# 摘 要

<span id="page-2-0"></span>智慧家庭的概念在物联网与人工智能技术的发展下逐渐被人们所接受,手势识 别作为一种新的交互方式,有其广阔的应用前景。

本设计基于手势识别相关技术实现了一个智慧家庭控制中心系统,通过对智慧家庭 相关设备的手势控制,以提供较好的操作便捷性和用户体验。本设计中的智慧家庭 控制中心系统主要基于 Java 开发,手势识别功能基于 Python 开发,主要实现了家庭 智慧设备的信息管理和维护、手势指令的识别与控制等功能。另外还开发了一个利 用手势控制智慧家庭设备的智能 Android 应用程序。本系统通过搭建一个卷积神经网 络来完成对各类手势的识别,最终实现使用手势控制家用智能终端的功能。本系统 软件主要分为设备信息管理、设备控制、手势采集、手势输入、用户管理五大模块, 本文对每个模块的设计与实现进行了详细的阐述和分析。通过对系统各模块的测试, 结果表明该系统符合用户需求。

关键词:手势识别 智慧家庭 设备控制

# **Abstract**

<span id="page-3-0"></span>The concept of smart home has been gradually accepted by people with the development of Internet of things and artificial intelligence technology. Gesture recognition, as a new way of interaction, has broad application prospects.

This design realizes a smart home control center system based on gesture recognition technology. Through gesture control of smart home related devices, it provides better operation convenience and user experience. The smart home control center system in this design is mainly developed based on Java, and the gesture recognition function is developed based on Python. It mainly realizes the information management and maintenance of smart home devices, the recognition and control of gesture instructions and other functions. In addition, an intelligent Android application using gestures to control smart home devices has been developed. The system builds a convolution neural network to complete the recognition of various gestures, and finally realizes the function of using gestures to control home intelligent terminals. The system software is mainly divided into five modules: device information management, device control, gesture acquisition, gesture input and user management. This paper describes and analyzes the design and implementation of each module in detail. Through the test of each module of the system, the results show that the system meets the needs of users.

**Key words :** gesture recognition smart family equipment control

## <span id="page-9-1"></span>1.3 相关技术

#### <span id="page-9-2"></span>1.3.1 卷积神经网络模型

<span id="page-9-3"></span>卷积神经网络是一种前向神经网络,它包括了卷积运算,并具有深度结构,是 一种典型的深度学习算法。卷积神经网络的研究始于二十世纪八九十年代,最早出 现的是时延网络和 LeNet-5;二十一世纪以后,由于深度学习的概念和计算机数值计 算设备的不断完善,卷积神经网络在计算机视觉、自然语言处理等方面得到了迅速 的发展。

CNN 是一种多层神经网络,每一层都包含若干个不同的神经元。CNN 的基本架 构由特征抽取和特征映射两个部分组成。C 层是特征抽取层,也就是卷积层。

#### 1.3.2 CNN 模型

<span id="page-9-4"></span>CNN 模型是一类在模式识别中得到广泛的应用。CNN 模型主要包括卷积层、池 化层和完全连接层,其中卷积层是 CNN 的核心部分,它可以通过学习不同的卷积核 来获得不同的特征。该算法充分利用 CNN 模型的优势,对数据进行反复优化,并对 手势的特征进行提取,以改善图像的识别效果。

#### 1.3.3 开发工具:Android Studio

 Android Studio 是一个安卓系统的整合开发工具,而 Android Studio 提供安卓 系统的开发与调试工具。Android 操作系统,比起 Eclipse 的启动速度、响应速度、 内存占用等问题,Android Studio 速度更快,更智能,还可以智能保存。

#### 1.3.4 开发工具:IntelliJ IDEA 编译器

IDEA 的全名是 IntelliJ IDEA,它是一个整合的 Java 编程语言的开发平台。IDEA 是 Java 开发的最佳工具, IDEA 倡导的是智能编程,以帮助开发者减少工作压力。

#### 1.3.5 MySQL 数据库

MySQL 是一种基于关系的开源数据库管理系统。MySQL 数据库采用了最常见 的结构式查询语言来管理系统的数据库。它支持各种不同的操作系统。由于该软件 具有操作简便、易用、性能稳定等优点,因此本系统将其用于系统中的数据库管理。

# 2 系统设计

#### <span id="page-11-1"></span><span id="page-11-0"></span>2.1 设计目标与原则

#### <span id="page-11-2"></span>2.1.1 设计目标

手势识别的智慧家庭控制中心系统的设计的目标主要有如下几点:

- (1) 通过设计出一个 app 训练手势,可以进行手势识别。
- (2) 该系统可以训练手势,在大量训练后,当获取拍摄的手势时,与数据集中 的手势进行比较,尽可能准确的识别手势。同时,为训练成果的检验提供了一种更 为智能化的方式。
- (3) 设定手势对应相应的命令,与模拟智能设备端相连接,实现智能设备端的 打开关闭等一些操作。

#### <span id="page-11-3"></span>2.1.2 设计原则

通过对系统用户的需求进行分析,可以了解到手势识别的智慧家庭控制中心系 统需要遵循的设计原则主要以下这几点:

- (1) 该系统是用于普通家庭用户的,所以界面设计应该遵循简单直观的原则, 方便使用者户可以在快速找到自己需要操作的部分。
- (2) 由于该系统的用户主要是每个家庭的普通用户,将来可以普及到每家每户。 因此,系统操作应该尽量的简单,方便大部分使用者能够快速掌握。
- (3) 该系统主要用于普通用户,为了提高用户的体验感,手势识别成功的概率 应该尽可能的高,真正为使用者提供便利,为使用者省时、省事。
- (4) 该系统目的是完成一个完整的手势识别的智慧家庭控制中心系统,所以智 慧终端设备应该尽可能的多,最好可以覆盖到普通家庭日常生活中的大部分智慧终 端。

# 结 论

<span id="page-35-0"></span>基于手势识别的智慧家庭控制中心系统主要针对快速、便捷的控制智慧家庭系 统问题提出了基于手势识别控制智能交互的解决方案,系统基于 WebStorm、IntelliJ IDEA 平台开发,实现了管理用户信息、控制设备、手势采集、手势输入等功能。 本系统优点如下:

- (1) 整个系统的网页设计简洁、直观,便于一般家庭用户的操作和使用。
- (2) 采集训练了大量的手势,手势识别成功率很高。
- (3) 与传统遥控器相比,手势识别无疑方便了非常多。智慧家庭最重要的是 可以为居民提供便利,将家庭生活智能化,其应用也就是人们身边天天做的事。而 手势识别设备简易,操作简便,十分适用于普通家庭。
- (4) 与市场上一些声控设备相比,手势识别使用人群更多,对一些聋哑人士 十分友好,也适用于语言不通的外国人。当然也适用与一些特定的场景,例如在深 夜不想吵到室友的休息时,做个手势就能打开台灯。而且手势识别成功率高于声音 识别的成功率,声音可能在用户不同状态时有细微差别,而手势识别就能完全避开 这个问题。

同时,本系统还有一些缺陷。例如目前手势训练的种类不够多,没有与真正的 硬件相结合。在实际的开发过程中,也遇到了不少问题。例如功能还不够全面,只 能控制一些电器简单的打开与关闭,暂时没有足够的精力、技术完成更多的功能设 置。本文所实现的并非是整个的智慧家庭。如果可以加入语音识别功能,相信一定 可以增加趣味性。在此之后,希望可以完善本系统,能够真正做出一个功能十分全 面的手势识别的智慧家庭控制中心系统。
