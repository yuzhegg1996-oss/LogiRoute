**Design and Implementation of Video Comment Sentiment Analysis System Based on Machine Learning**

## 摘 要

<span id="page-2-0"></span>基于机器学习的视频评论情感分析系统旨在通过先进的人工智能技术,深入挖 掘视频特征与用户偏好之间的关系,然后区分用户评论的情感倾向以提升视频推荐 系统的精准度。这个系统会收集很多视频相关的数据,比如用户看完视频后写的评 论、打的星级评分,还有视频本身的类型等信息。收集到数据后,系统会把它们整理 到一起,并按照不同的类别进行分类,方便后面分析使用。利用 OpenAI 的 API 对评 论内容进行情绪分析,然后判断情绪值状态为积极、中性或消极,将结果作为评论的 等级,把这些信息保存到数据库的 Comments 模型中,实现对视频评分的准确预估。 通过用户对视频的评分,能了解不同用户对视频评论情感的情绪值判断。实验结果表 明,这个系统可以明显提高评分预测的准确程度。它比传统的机器学习方法更好,能 够为视频推荐提供有力的数据支持。它不仅能优化用户对不同视频的体验,还能为视 频产业的内容分发和市场营销策略提供科学依据。

关键词:视频评论 情感分析 视频特征 用户偏好 推荐系统

## **Abstract**

<span id="page-3-0"></span>The video comment sentiment analysis system based on machine learning aims to deeply explore the relationship between video features and user preferences through advanced artificial intelligence technology, and then distinguish the sentiment tendencies of user comments to enhance the accuracy of video recommendation systems. This system collects a large amount of video-related data, such as comments written by users after watching videos, star ratings given, and the type of the video itself. After collecting the data, the system organizes them together and classifies them into different categories for convenient analysis later. The OpenAI API is used to analyze the sentiment of the comment content, and then determine the sentiment value status as positive, neutral, or negative, which is used as the comment grade and saved in the Comments model of the database to accurately predict video ratings. Through users' ratings of videos, the sentiment value judgments of different users on video comments can be understood. Experimental results show that this system can significantly improve the accuracy of rating predictions. It is better than traditional machine learning methods and can provide strong data support for video recommendations. It not only optimizes users' experiences with different videos but also provides scientific basis for content distribution and marketing strategies in the video industry.

## 1 绪 论

## <span id="page-6-1"></span><span id="page-6-0"></span>1.1 研究目的和意义

#### <span id="page-6-2"></span>1.1.1 课题背景

当前阶段,迎来黄金发展期的短视频行业正经历着显著变化,互联网技术作为重 要推动力量发挥着作用,视频数据总量呈现出指数级增长趋势。值得关注的是,视频 评论数据这一特殊存在形态,其潜在价值尚未得到充分开发[\[1\]](#page-40-1)。普遍存在于各类影评 平台、社交媒介的用户评分行为,已经成为当代观众的日常实践。这些数值化评价并 非简单数字排列,实则承载着多维度的观影体验:影片偏好程度的直观体现可见于 此,质量评估的客观标准隐藏其间,观影预期达成度的真实反馈亦包含其中。影院管 理者对影片市场潜力的准确预判尤为关键,排片决策的科学性由此决定[\[2\]](#page-40-2)。精准把握 观众反馈信息的重要性已得到行业各环节主体的广泛认同。

传统的分析手段面对复杂评分数据时存在明显局限性。演员的精湛演技、导演的 独特风格、剧情的紧凑节奏、特效的震撼运用等维度相互交织共同影响观众评分这一 现象,传统方法无法给出明确解释。影评文本中或激昂或委婉的情感倾向与评分之间 存在复杂关联性,传统工具同样难以准确识别。实例表明,视频优化改进所需的针对 性建议无法通过这类方法获得。

在视频评分分析领域显现出革新性潜力的,正是当前备受关注的深度学习技术 化应用。情感特征的精细化提取过程在此类模型中得以实现,与用户评分数据间的关 联性映射被有效建立[\[3\]](#page-40-3)。

两种技术路径的协同整合过程中,得以构建的是混合模型之性能表现,超越传统 分析范式已被观察到[\[4\]](#page-40-4)。视频评论数据内隐含的深层规律,系统性揭示由此实现。传 统人工分析方法难以捕捉的那些复杂关联,有效建模在此过程中获得实例支持。

当前视频市场呈现高度饱和态势,各相关利益方迫切需求智能化解决方案者众, 以期实现视频评论数据价值的深度挖掘化。贯穿视频内容生产全周期化,此类技术可 提供关键决策支持作用。基于机器学习架构的视频评论情感分析系统实例表明该技 术正逐步满足市场对于数据驱动型决策的急迫性需求。

站在理论研究的视角观看,不能忽视它的重要性。一方面,得以拓展的是深度学

习技术于文化娱乐领域的应用边界,与视频艺术创作实践产生深度融合现象,具有开 创性的是构建完成视频评分分析理论体系这一工作,为未来学术研究提供了方向性 指引。值得注意的是,通过运用深度学习这一研究工具,能够深入解析隐藏在用户评 分与影评文字中的情感倾向与认知模式,实例表明这为视频传播学等相关学科的理 论发展注入了新的动力源泉。由此可见,相关理论框架因此获得了更为丰富的内涵支 撑。

#### <span id="page-7-0"></span>1.1.2 目的和意义

视频评论情感分析系统的构建依赖机器学习方法,核心目标是对用户情感倾向 性的深度解析。视频平台上海量用户不同的评论数据,由系统实施自动化处理与识 别,然后被划分为积极、消极及中性三类情感维度[\[5\]](#page-40-5)。创作者群体与平台运营方能够 很好的看到不同的方面,表明对内容策略的调整具有指导价值。

用户体验的优化构成该系统的重要设计初衷。创作者可根据观众反馈实施内容 调整策略,实例可见内容质量与用户满意度均呈现提升态势。平台管理者方面,情感 趋势的可视化分析功能被提供,舆论环境的监控由此变得更为高效。潜在情感危机的 识别与应对机制得以建立,实例显示这对社区生态维护产生积极作用[\[6\]](#page-40-6)。

更值得关注的是视频社区氛围的良性发展可能性。通过情感分析技术的引导作 用,积极讨论氛围的营造成为可能。用户参与度指标与社区凝聚力参数均获得显著改 善,由此可见该系统的社会价值不容忽视。

在技术融合层面具有突破性意义的是本系统的实现过程。大数据方法与人工智 能技术的交叉应用开创了视频内容理解新范式。评论数据的深层特征被有效挖掘,创 作者据此获得相对精确的反馈信息[\[7\]](#page-40-7)。贴近用户需求的高质量内容生产成为现实可能, 实例证明这对行业发展具有推动作用。

社会应用层面的延伸意义同样不可低估。舆论引导效能的提升与品牌形象的维 护构成典型应用场景。公众情绪波动的监测能力为决策制定提供数据支撑,负面情绪 的防控机制由此建立。社会稳定和谐的维护目标获得新的实现路径,实例表明这具有 重要的社会治理价值。视频评论情感分析系统的研发实践昭示着技术发展的新方向。 不仅视频内容理解领域取得实质性突破,更在平台运营、创作生态及社会治理等多个 维度产生深远影响。

## <span id="page-8-0"></span>1.2 国内外研究现状

#### <span id="page-8-1"></span>1.2.1 国外研究现状

在国外,关于情感解析技术的研究热度正持续攀升。Web 2.0 阶段初期便可见端 倪,当时用户生产内容呈现爆发式增长态势,使得情感解析逐渐演变为自然语言处理 学界重点关注方向[\[8\]](#page-40-8)。2006 年起,由国际文本检索会议 TREC 主导,每年例行举办 涉及情感解析的评测项目。NTCIR(日本国立情报学研究所信息检索测试集)与COAE (中文意见分析评测)等系列评测活动亦相继涌现。为相关技术发展搭建了重要平 台。实例表明,这些标准化评测体系的建立不仅规范了研究范式。更促进了学术成果 的广泛传播。针对影视评论的情感研判领域,西方研究者已获得诸多突破性进展。采 用支持向量机、朴素贝叶斯等传统机器学习方法。对评论文本实施情绪极性划分取得 了理想效果。值得注意的是,随着深度学习方法兴起,诸如循环神经网络和卷积神经 网络等新型模型被频繁引入该领域。由此可见,这些创新性研究既为在线视频服务商 提供了技术支撑[\[9\]](#page-40-9)。

#### <span id="page-8-2"></span>1.2.2 国内研究现状

在国内,情感分析技术的研究开端相对较晚,不过近些年来也收获了较为十分突 出的发展成果,随着中文情感分析评测任务 COAE 的举行,有越来越多的学者着手 关注中文情感分析技术的研究工作以及实际应用情况。在视频评论情感分析方面,国 内学者也积极探索适合中文语境的情感分析方法[\[10\]](#page-40-10)。他们利用中文分词技术、情感 词典等资源,结合机器学习算法,对视频评论进行情感分析,取得了一定的成果。当 前阶段而言,视频评论情感分析系统的机器学习化构建研究呈现快速发展态势[\[11\]](#page-40-11)。 同时,国内学者也积极借鉴国外的先进技术和经验,推动中文情感分析技术的不断创 新和发展。

#### <span id="page-8-3"></span>1.2.3 国内外现状总结

在视频评论情感分析领域的研究成果方面,国内外学者均取得了一定进展,然而 诸多挑战性问题仍待解决[\[12\]](#page-40-12)。特征的有效提取与表征化处理成为首要难题,大规模 高质量标注数据的获取亦存在显著困难性。由此可见,该领域的深入发展需要全球研 究者的协同性探索与持续性创新[\[13\]](#page-40-13)。

技术迭代的持续化推进与应用场景的多元化拓展背景下,情感分析技术的重要

性将日益凸显。个性化服务体验的智能化提升将成为视频平台的重要发展方向,实例 表明该技术的应用前景具有广阔性[\[14\]](#page-41-0)。

#### <span id="page-10-2"></span>2.1.1 经济可行性分析

在可行性经济维度层面考量,较高成本属性显现于此系统架构中。庞大市场体量 构成视频产业领域,数据分析需求呈现持续增长态势。耗时费力特征普遍存在于传统 分析方法之中,用户偏好与视频特征间复杂关联难以被准确捕捉。自动化处理机制则 被机器学习驱动的评论情感分析系统所采用,大规模视频数据的高效分析得以实现, 数据分析成本由此获得显著降低效果。

精准市场洞察与用户画像生成能力具备于该系统内部,视频制作机构、内容发行 方及放映终端等主体可从中获益。营销策略优化与创作计划调整成为可能,实例表明 票房收益提升与品牌影响力强化随之产生。经济效益最大化目标通过系统应用达成。 由此可见,经济视角下的系统设计方案具有切实可行特性。

#### <span id="page-10-3"></span>2.1.2 技术可行性分析

技术可行性维度上,Django 框架与 Vue.js 技术之组合为系统开发工作提供了强 有力的支撑性保障。具有高级 Python Web 框架特性的 Django,其优势体现在开发速 度快速化、安全性能优越化以及扩展便捷性等方面,特别适用于构建复杂程度较高的 后端服务系统。数据存储功能实现、处理流程执行以及分析任务完成,通过 Django 框架得以高效运作,为深度学习模型提供着可靠的数据支撑作用。

轻量级特征显著的 Vue.js 前端框架,则配备了丰富的组件资源库与灵活度极高 的数据绑定机制,这使得前端界面开发工作呈现出效率提升态势与便捷性增强趋势。 用户界面设计方面,借助 Vue.js 技术能够构建出直观性强、操作简便的交互系统实 例。视频评分信息查看、影评内容浏览等功能的实现变得简单易行,用户参与视频推 荐流程和评分过程亦获得便利性保障。

## <span id="page-32-1"></span>5.1 系统测试的概念

系统测试是系统开发的重要步骤,是针对整个项目进行的测试,目的是为了验证 系统是否满足了需求规格的定义,找出与实际要求相差不符的问题,加以改进从而呈 现出更加完美的系统,理论上说不能全部的解决报错,但错误越少,系统运行的就越 流利,用户使用更方便。系统的测试包括对其功能的测试,安全的测试,可靠性的测 试。测试的实施依据在于各模块功能性特征及其间相互协同作用关系的评估存在功 能联系的模块间协同性为测试重点,各模块功能属性被充分考量,通过一次又一次的 进行测试,发现系统中的问题,在对问题进行反复的修改,是系统更加的完善。还要 在测试和修改系统的过程中吸取经验,反思自己为什么在系统中会出现这样的问题 那个环节出现了错误。

软件系统测试是在人工与自动化的手段来测试软件系统的过程测试方法主要有 黑盒测试与白盒测试。

黑盒测试又称之为功能性测试,黑盒测试是软件测试中常用的一种测试方法,它 可以为用户测试在用户角度下的表现与功能问题,因此可以提高软件的用户体验感 与系统质量。通过黑盒测试,可以使系统中的功能缺陷,边界问题等暴露出来,可以 是测试人员不用深入了解程序的条件下发现解决这些问题。

白盒测试又叫做结构性测试,通过检查对软件系统内部的逻辑结构,和对软件中 的逻辑路径进行覆盖测试遵循一个模块的独立路径至少测试一次,保证其结构有效 性等原则进行测试以确定实际运行状态与预期状态是否一致。

本文的系统主要用黑盒测试,使用黑盒测试,检验出系统是否有功能错误,是否 有功能遗漏,是否能正确的接受数据,并输出正确的结果,是否有软件程序初始化和 停止等方面问题,保证系统的正常运行。

在较大规模的代码单元测试中,相较于白盒测试方法,更高的效率被黑盒测试所 展示。实现的细节无需测试人员掌握,特定编程语言的知识亦不必要。基于用户视角 的测试过程,理解与接受度均较为理想。规格不一致或存在歧义的问题,实例表明能 够通过此种方法有效暴露。

## 结 论

<span id="page-38-0"></span>该系统在人工智能技术的支持下实现了对视频评论的情感分析,同时结合视频 特征进行用户评分统计的操作。视频推荐的准确性提高,平台与内容创作者对用户的 偏好的进行更深一步的理解。

在系统架构层面采用前后端分离的模式,由 Django 框架负责后端数据逻辑处理 工作,交互界面通过 Vue.js 技术实现构建。视频数据与评论信息的存储任务由 MySQL 8.0 数据库完成。评论情感分析功能借助 OpenAI 接口实现,数据可视化部分采用 ECharts 工具处理,而 Python 编写的爬虫程序从豆瓣等平台获取评论原始数据。

主要功能模块包含视频名称、评分及用户评论的爬取功能,重复内容的去除与数 据清洗工作同步进行。通过 OpenAI 接口为每条评论标注"积极/中性/消极"情感标 签后,结合视频类型等特征完成用户评分预测建模。情感分布情况与评分变化趋势用 图表呈现,便于用户更加直观的理解。管理员权限设置方面包含评论审核、视频信息 更新以及数据分析用数据导出等功能。

由测试结果表明情感分类准确率获得了显著提升,推荐视频点击率有明显的增 长。与传统的方法相比系统在评分预测误差率方面有所降低。

综上所述,该系统基于 Django、Vue.js 和 MySQL 技术架构实现视频评论情感分 析具有可行性。对于视频平台来说,可以用这个系统优化推荐算法,让用户更容易找 到喜欢的视频;创作者能通过评论情感了解观众反馈,调整创作方向;广告商能根据 情感分析结果,把广告投给更合适的目标人群,帮助各方做出更好的决策。
