# 摘 要

<span id="page-2-0"></span>现代技术的快速发展已经改变了人们的工作和生活方式。人们对生活的品质以 及日常安全要求更高。一般的门禁系统采用感应智能卡识别,存在着使用不方便且需 要随时佩戴等问题。因此,智能门禁控制系统被广泛应用于人们的日常生活中。利用 手势对电子产品进行命令操作已经成为了一项关键技术。本篇论文研究了一个基于 手势识别的门禁系统,但是手势可能会被模仿,应用场景被限于并不是很严格的门禁 系统上。手势具有一定的安全性,同时也有便捷性,有它适用的场景,所以本文将手 势识别作为门禁管理系统的一个功能点,来完成一些控制门禁操作,而不做身份验 证。本系统根据用户需求分为门禁信息录入、门禁口管理、手势控制、密码解锁、日 志管理五大模块,对每个模块进行了分析设计与实现。通过对系统各模块的测试,结 果表明该系统符合用户需求。

# **Abstract**

<span id="page-3-0"></span>The rapid development of modern technology has changed the way people work and live. People have higher requirements for the quality of life and daily safety. The general access control system uses inductive smart card identification, which is inconvenient to use and needs to be worn at any time. Therefore, intelligent access control systems are widely used in people's daily life. Using gestures to command electronic products has become a key technology. This article is to study the access control system based on gesture recognition, but gestures may be imitated, and the implementation scenario limited to the access control system. that is not very strict. Gestures have certain security and convenience, and have their applicable scenarios, so this article uses gesture recognition as a function point of the access control management system to complete some access control operations without authentication. According to user needs, the system is divided into five modules: access control information entry, access control gate management, gesture control, password unlocking, and log management. Each module is analyzed, designed and implemented. Through the test of each module of the system, the results show that the system meets the needs of the users.

# 绪论

<span id="page-5-0"></span>随着人民的经济和生活水准的提高,人民对生活品质的重视程度也在不断增加,不仅对房屋住宿有要求,对安全智能的门禁系统也更加关注。传统的门禁系统采用钥匙、密码等物理开锁方式开启,这种方式不够安全方便,而且户主如若忘记携带钥匙或丢失钥匙也比较麻烦。计算机技术在不断发展,我们在智能门禁控制系统这方便的研究和开发也趋于成熟。

本文是研究基于手势识别的门禁控制系统,但是手势可能会被模仿,应用场景被限于并不是很严格的门禁系统上。手势具有一定的安全性,同时也有便捷性,有它适用的场景,所以本文将手势识别作为门禁管理系统的一个功能点,来完成一些控制门禁操作,而不做身份验证。在人员经过门禁时通过手势识别获取经过人员的手部动作,将识别结果与数据库中的手势信息进行查询比对,如果匹配成功则可以

执行相对应的门禁操作。因此,在门禁控制系统中采用手势识别技术具有较大的研 究价值和市场空间。

本系统将手势识别控制与传统门禁控制系统相结合,顺应时代变化,体现了 "互联网+门禁"思想,降低了人工成本,提高了管理效率。

# 1 系统开发背景

# <span id="page-7-1"></span><span id="page-7-0"></span>1.1 选题背景

随着计算机视觉技术的迅速发展,传统的人机交互技术(如键盘、鼠标等)逐渐 无法满足人们的日常需求,许多研究人员已经开始致力于开发新的人机交互技术, 例如人脸识别和手势识别等。在这些新型人机交互技术中手势识别非常重要,它主 要分为静态手势识别和动态手势识别。利用摄像头这类视频输入设备来对用户的手 部动作进行检测、追踪、识别。基于视觉的动态手势识别是最直观自然的人机交互 方式之一。视觉手势识别也应用于各大领域,它给人机交互带来了一种全新的交流 方式,用户和计算机之间只需要使用手部动作就可以进行交流[\[1\]](#page-44-1)。

智能门禁考勤系统也广泛应用在生活中,其功能也在逐步完善。以往的门禁控 制和考勤一般采用密码、指纹识别、人脸识别,现在则可以将三者结合成一个智能 门锁来控制门禁系统。当门禁系统的安全性、智能性都普及后,就可以大大提高其 效率,从而降低人工成本[\[2\]](#page-44-2) 。

智能门禁系统在大学实验室中也有应用,例如基于图象处理技术的门禁管理系 统主要是为了提高实验室的使用率,同时降低高校实验室管理人员的工作负担。随 着计算机技术的飞速发展,我们已经能通过采集、分析、比对人脸图像来对进出人 员进行身份核实。此系统是通过人脸图像识别并结合 51 单片机控制技术,以此来 完成进出人脸图像的采集、分析和识别,自动开关门及自动记录人员进出情况,能 够有效地解决实验室管理问题[\[3\]](#page-44-3) 。

目前的门禁系统安全性较低,操作和管理都比较可能,因此通过树莓派的人脸 识别和手势识别门禁系统来解决这一问题。把静态的人脸识别和动态的手势识别相 结合来确定是否允许用户通过,这大大提高了门禁的安全性,更加方便快捷。人脸 识别主要是定位和检测人脸,然后进行图像的预处理,提取人脸特征,最后进行识 别。手势识别则是先建立好训练集,开启手势交互视频,然后对手部动作进行定 位,提取手势轮廓进行手势识别[\[4\]](#page-44-4) 。

传统的机械锁、密码锁等物理控制系统已经无法适应新型互联网时代的快节奏 了,尤其是面对大型的开放式社区,更加难以管理。针对物联网门禁系统的技术和 需求,提出了一种开放式智慧门禁控制系统的设计方案。此系统更加具有开放性, 可拓展性,还具备较强的网络通信能力且可以远程操控,更符合现代社会的需求 [\[5\]](#page-44-5)。

提到网络通信话题,信息入侵攻击最为影响网络性能,基于生物特征的认证已 经成为信息安全的组成部分,生物特征主要用于识别和认证。用于门禁系统的手部 图像认证可以解决这一问题,使用手部图像来定义一个参考物,然后进行验证识别 [6]。

#### <span id="page-8-0"></span>1.2 选题意义

门禁系统是居民安全生活的第一道保障,需要为居民提供更方便快捷的居住环 境,在实际生活中具有很大的应用价值,其市场需求也很大。在现代技术不断更新与 发展今天,传统的门禁系统已经无法满足我们对普遍、安全、便捷等高质量服务的要 求。因此需要一种新的设计思路来改变这一现状。

众所周知,在我们日常的交流方式当中,除了自然语言,肢体语言也占有相当大 部分的比例,其中,手势动作一目了然,可以很直白地传递信息,在我们的日常交流 中更是起着关键性作用。在人机交互领域,按照手势所表达内容的差别,人们所使用 到的手势可以分为控制类手势和对话类手势。控制类手势指人们具体的指令,主要在 单向交互的机械领域使用,而对话类手势指人们自然的交流行为,使用它可以实现人 机的双向交互[7] 。静态手势只具有空间特性,动态手势是静态手势与时间的组合,其 动作表达具有时间持续的属性,包含有时、空两种特性[8],本课题中摄像头识别手势 动作将用到。手势识别是一种新兴技术的产物,涉及学科广泛,应用场景多样,对手 势识别的研究具有重要价值。在这个"互联网+"思维深入人心的时代,以"互联网 +门禁"作为设计思想,可以有效地解决传统门禁的一些弊端,并且可以提高门禁系 统的可靠性和可扩展性[9] 。

# 2 用户需求和技术分析

<span id="page-9-0"></span>基于手势识别的门禁控制系统软件主要实现对居民小区应用场所内多个需要管 控的门进行信息化、网络化的控制与管理。本系统的定位是实现居民小区场景下的门 禁系统,使其可以很好的成为小区居民安全生活的第一道保障,并通过手势识别控制 增强门禁系统的方便性,为居民提供更方便快捷的体验。

#### <span id="page-9-1"></span>2.1 功能性需求分析

该系统主要采用权限分类管理,即智能门禁控制系统预先设置了两个使用者,一 个是管理员,另一个是社区的住户。不同的身份角色可以使用不一样的功能,这样便 于对系统进行分级管理,使管理在层次上更容易管理、更安全可靠。还可以对重要数 据进行分类和存储,以确保数据安全。不同的角色有不同的权利,这样整个社区就能 以更人性化、更安全的方式进行管理。

#### <span id="page-9-2"></span>2.1.1 用户功能

住在小区内的住户,可以自由通过小区的大门和自己所住楼层的门禁。其功能如 下:

- (1)在手势控制部分,可以通过执行预设的手势控制操作,来完成呼叫门卫室、 开门、锁门等操作;
- (2) 居民用户通过门禁口时,可以输入密码解锁,密码正确则可以给予通过, 反之,则不予通过。

#### <span id="page-9-3"></span>2.1.2 管理员功能

管理员对基于手势识别的门禁控制系统拥有最终的管理权限,包括对管理系统 信息的输入,设置各种参数和历史记录的查询等。其功能如下:

- (1) 管理员可以录入门禁相关信息以及系统配置数据;
- (2)对门禁口进行管理,可以浏览和查询门禁口信息,查询门禁口的放行口令, 在出现问题后及时维护门禁口信息;
  - (3) 管理员可以设置门禁口的手势控制动作和正确通行的门禁密码;
- (4) 管理员可以记录和查询门禁放行操作,也可以查询其他门禁操作的历史记 录。

#### (1) Python 语言

Python 语言是一种面向对象、直译式计算机程序设计语言,它语法简捷清晰, 可读性强,并且它还是一门开源的语言,有着强大的资源库的支持。除此之外,Python 语言还具有着简单易学、资源丰富、扩展性强等特点,使用非常便捷,广泛应用于人 工智能领域[\[10\]](#page-44-6)。

#### (2) SpringBoot 后端开源框架

SpringBoot 框架是继承 Spring 框架优点的一个开源应用框架,然后它创建和开 发 Spring 应用程序的过程也被简化。以前的项目需要大量的配置文件,劳动强度太 大,影响开发效果。而 SpringBoot 的诞生很好地解决了这些问题,它的出现大大提 高了开发者的效率[11] 。

#### (3) Vue 前端开源框架

早期的网络前端框架只包含 HTML(管理页面结构)、CSS(负责页面样式)和 JavaScript(控制用户与页面的交互)这三个部分,三者分工明确[\[12\]](#page-44-7)。然而,这种模式 有很大的局限性,因为前端只负责页面的设计,而大部分的功能必须通过后端来实 现。随着网络应用的发展,前端的功能越来越多,这种模式的缺点也越来越明显,许 多优秀的前端框架例如 Vue.js 开始流行。

Vue 是一套用于构建用户界面的高级 JavaScript 应用程序,Vue 核心库更方便我 们进行学习,Vue.js 的目标是尽可能使用简单的 API 实现响应数据的绑定和组合的 视图组件[\[13\]](#page-45-0)。

#### (4) OpenCV 框架

OpenCV 框架它轻量级且高效,通过使用 Python、MATLAB 等语言接口来实现 计算机视觉方面和数字图像处理的大部分通用算法。一般的计算机视觉软件可能不 兼容、不稳定且耗费高,而 OpenCV 致力于成为可以简化计算机视觉程序和解决方 案的开发的标准 API。

OpenCV 是一个用于图像处理的实用函数库,调用这个库可以通过处理图像和 视频来完成各种任务,包括显示摄像头输入的信号以及使用机器人识别现实生活中 的物体等。历经几个版本的升级,如今 OpenCV 已经成为最重要的图像处理工具之 一。由于 OpenCV 库本身是开源的,所以,用户不仅可以直接调用 OpenCV 中现有 的函数,还可以自己编写函数,添加到 OpenCV 库中,这样就能够有针对性地进行 软件开发,提高开发效率[14] 。

#### (5) MongoDB 数据库

MongoDB 的出现是为 Web 应用提供数据存储解决方案。它可以存储更加复杂 的数据类型,因为该数据库所支持的数据结构比较松散。MongoDB 使用文档存储数 据,文档采用 BSON 数据格式,是一种二进制编码的 JSON,因此具备 JSON 的所有 功能,且更易于在网络传输和存储[\[15\]](#page-45-1)。除此之外,BSON 还提供了 JSON 所不支持的 数据类型比如日期类型。所以本系统采用 MongoDB 数据库。

#### (6) 前后端分离模式

前端和后端解耦简单地说就是把一个应用程序分成两部分,前端应用程序和后 端应用程序以 JSON 格式进行通信。前端和后端各司其职,只关注各自领域的工作, 然后用约定好的接口进行数据的调用,这种开发模式是现在所需要的,可以减轻程序 员的工作量,也能使开发程序更加高效。

# 6 研究成果与社会和经济的关系

<span id="page-41-0"></span>智能门禁系统是指区别于传统机械门禁的基础上改进的,在用户安全性、识别、 管理性方面更加智能化便捷化的一个管理系统。本论文针对社区居民对门禁控制系 统的便捷性、安全性、稳定性的需求,研究和开发了基于手势识别的门禁控制系统。

在社会层面来讲,随着互联网技术的发展以及近两年"新冠疫情"爆发带来的 影响,我们一直倡导"保持距离少接触"的观点,本文将手势识别技术应用在门禁管 理系统上,依靠手势识别控制来完成锁门、呼叫门卫室等操作,很好地减少了接触其 他人的概率。本文将智能门禁系统的核心理念定位在"安全"上,既符合当前阶段需 要遵循的疫情防控要求,也顺应了当前阶段门禁控制系统迅速发展的时代潮流。

在经济层面来讲,传统的门禁管理系统采用 ID 卡,利用 RFID 技术传递用户授 权管理不是很方便,过于依赖 ID 卡等附加硬件,使用过程是不太便捷的,还要担心 ID 卡的丢失、损坏带来的不便。并且传统门禁系统也更加依赖人力,需要消耗更大 的人工成本,耗费大量的财力、物力、人力。智能门禁管理系统不仅更加方便安全, 成本也更低,还实现了智能化、自动化,符合当前社会的需要与发展。

从可持续发展层面来讲,该智能门禁系统既满足了当代人发展的需求,又不危及 后代人满足其需求。并且依靠手势识别技术就不需要传统的身份卡等成本消耗,更加 环保,所以该系统发展也协调了环保与经济发展之间的关系。

# 结 论

<span id="page-42-0"></span>智能门禁系统是未来社区使用的发展趋势,也是智慧城市的基本组成单位和智 能家居进行联动。通过学习研究,我们能够更好地掌握相关技术,并将其与社会住房 用户的实际需求联系起来,对基于手势识别的门禁控制系统设计进行了研究。

本论文首先介绍了门禁控制系统的发展现状以及需要用到的相关理论和技术基 础,论述了基于手势识别的门禁控制系统的总体架构,并重点分析了门禁控制系统的 功能需求和软件总体架构设计。在此基础上,从系统功能层面,详细阐述了门禁管理 系统各功能模块的设计与实现。最后,对研发完成的管控软件进行各方面的详细测 试。测试结果表明,本论文研发的门禁管理系统符合社区居民用户需求。

将手势识别技术应用在居民社区门禁控制系统上其实是有一点问题的,手势可 能会被模仿且不具有唯一性,所以不能很好地达到"安全"这一性能,本文是将手势 识别作为门禁控制系统的一个功能点,来完成控制门禁的操作,不将其作为身份验 证。我觉得在未来的研究设计中,可以着重在这一方面,寻找手势识别唯一性的解决 方法,将手势识别与门禁管理系统更加贴合。其次,我们主要研究的是智能门禁控制 系统软件方面,对于硬件方面还没有得到解决,在以后的时间里需要对这方面进一步 研究。此外,本文所讨论的智能门禁系统的手势识别部分也是当今的一个热门话题和 研究重点,伴随着技术的进步和发展,相关技术的开发必须朝着更全面的方向发展。 市场需求是技术发展的重要驱动力,不同技术的结合将使其更贴近实际需求,有助于 发现问题,促进技术改革和创新。
