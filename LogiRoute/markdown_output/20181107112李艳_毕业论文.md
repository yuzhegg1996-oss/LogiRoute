# <span id="page-12-0"></span>1.4 相关技术

#### (1)Spring Boot 框架

Spring 框架作为 Java 运用于平台中的一个开源性应用的架构,并提出了容器具 有反转控制特征。提供了一个可以利用 Java 反射机制对所有 Java 对象进行统一的构 建与管理的方式。通过 Spring 框架可以使用 XML 文件或者类中的某个 Java 注解来 完成对象配置,同时开发人员也能够使用依赖进行查询或是使用依赖去注入信息来 得到新对象,在开发过程中可以很大幅度的去提升开发者的开发程序工作效率。

#### (2)MyBatis 框架

使用了持久层的、轻量级的半自动化 ORM 架构,将所有 JDBC 操作都进行了封 装,还有设置查询参数和获取结果集的操作,取消了对大量的 JDBC 代码和手工设 定的参数的和结果集的检索,使用了简单注解进行配置和原始映射,没有对程序的 现有设计产生任何的负面影响,而且灵活性很高,并且支持自己去定义 SQL。

#### (3)MySQL 数据库

MySQL 是一种关系型数据库的管理系统,在不同的表里面存储不同的数据,不 是把系统所有的数据去放进同一个仓库里面,运用这种方式进行处理提升了速度还 让灵活性变高。运用常用非常准确语言进[行数据库访](https://baike.baidu.com/item/%E6%95%B0%E6%8D%AE%E5%BA%93/103728)问,它体积小、速度非常快、总 体来说拥有的成本低。

# <span id="page-37-1"></span>4.1 测试目的与技术

#### (1)测试的目的

基于手势识别的购物交互系统测试是项目完成中不可缺少的一个部分,有着十 分大的意义,完成该基于手势识别的购物交互系统的开发工作,对系统进行个个功 能的测试,来保证系统按照之前期待的方向进行运行,从而实现所有的目标功能。 通过对这一整个系统来进行安排测试,去尽量全面的发现该基于手势识别的购物交 互系统中所存在的一些缺陷,从而去保证了购物系统的质量。在测试过程中,根据 该基于手势识别的购物交互系统进行测试用例的编写,严格按照测试用例执行。

#### (2)测试的技术

软件测试可以用很多的方法,测试常见分为对代码测试的白盒测试、对功能测 试的黑盒测试以及灰盒测试。白盒测试也被称之为结构测试,它进行测试是对应源 代码中的文件,细致对完成过程进行检查,非常全面的去检测整个系统程序的代码 中结构有没有出现问题。黑盒测试也被叫做功能测试,是写测试用例再去测试系统 程序有没有在功能的问题。在本次基于手势识别与购物交互系统使用黑盒测试去测 试,按照使用者的角度去测试会不会遇到什么 bug。本次基于手势识别的购物交互系 统测试范围是用户的注册和登录、添加购物车、订单生成、后台的管理商品和后台 的管理订单这几个方面进行黑盒测试,这次我使用了等价类划分,边界值分析方法 去对这个购物网站进行测试。

<span id="page-42-0"></span>随着科学技术发展,手势识别应用在越来越多的行业,本次研究的课题是基于 手势识别的购物交互系统,在互联网的发展下,使用网上购物的人群也越来越广泛。 传统的买卖商品在实体店面内,用户的覆盖面很窄,客源固定,线上的电子商务在 近几年的快速发展改变了这一现象,人们可以在网上进行购物,不再局限于所居住 地方店面里面的商品,通过快递将商品去运送到全国各个地方,改变了传统的去店 里购物的这一方式。通过手势识别添加进购物系统后,可以增加用户使用本系统进 行购物的乐趣。

本次课题通过将手势与购物相结合,改变了之前的使用方式,促进了人们对购 物系统固有的认知。发展的前提是需要不断的创新,手势与购物结合进行创新,这 将推进购物网站的不断进步发展,推进社会的发展,而我也相信,此次手势与购物 相结合完成的购物系统,未来不断的去完善后,用户在使用该基于手势识别的购物 交互系统的时候可以通过做手势去完成购物的全部流程,使得更加便利。

在未来的科技之中,伴随着手势识别的不断发展,技术会越来越成熟,所应用 的地方也会越来越广泛。将手势识别与购物系统进行结合,改变一般的商城设计模 式,促进用户使用这类型的交互网站,如此以来,我们的生活也会越来越富有科技, 从而去带动线上购物这类经济的发展。

# 结 论

<span id="page-43-0"></span>手势作为人类在社会上一直存在的重要肢体的语言,将手势运用到购物系统之 中,通过识别手势,去完成对应的功能,手势识别与购物相结合后,用户通过做出 手势来完成商品加入购物车这个功能。手势识别与购物车相结合,做出一个基于手 势识别的购物交互系统,改变了传统的线下购物模式,它具有界面简洁,使用方便, 功能清晰的特点。在该基于手势识别的购物交互系统的设计中,使其具有简洁性与 实用性,用户在学习操作过程后,可以很快学会如何使用。

采用了与百度云的接口进行调用,调用手势识别的 API 后,上传到百度云的手 势识别后,判断手势是不是指定的手势,识别手势后返回值,判断返回值,如果手 势正确就加入购物车,从而实现手势识别与购物系统的交互。

基于手势识别的购物交互系统中创新的地方,是与一般的购物网站不同的是, 它将手势识别运用到购物之中,通过手势识别可以将商品添加进入购物车里面。这 次设计本基于手势识别的购物交互系统时,选用 Java 语言作为开发语言,采用 Maven 构建项目,运用了 Spring Boot 框架,数据库使用 MySQL,手势识别部分通过使用 百度云第三方实现。基于手势识别的购物交互系统中的 Android 端使用了 C/S 结构, 后端使用了 B/S 结构。

对系统大概的框架确定后,去实现各个功能模块,完善系统的功能。在最后测 试中,通过对该基于手势识别的购物交互系统的各个功能进行测试,测试中发现了 系统存在一些小问题,但我相信在今后的不断修改和完善后,这个基于手势识别的 购物交互系统将会变得越来越完善。
